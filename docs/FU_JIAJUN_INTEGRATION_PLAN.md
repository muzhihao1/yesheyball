# 傅家俊十大招训练体系集成方案

**版本**: V2.1
**日期**: 2025-11-10
**目标**: 将52集系统训练与傅家俊十大招融合为统一的8级成长体系

---

## 📊 整体架构

```
┌─────────────────────────────────────────────────────┐
│          耶氏台球训练系统 V2.1                        │
├─────────────────────────────────────────────────────┤
│                                                      │
│  1️⃣ 闯关系统 (levels.tsx)                           │
```

### 阶段二: 前端重构 (3-5天)

#### 2.1 重构 `tasks.tsx` → `fu-training.tsx`

**主要改动**:
1. 移除简单计时器模式
2. 调用 `/api/training/levels` 获取8级数据
3. 展示训练单元列表，每个单元可点击查看详情
4. 训练模式改为:
   - **学习模式**: 查看theory、steps、tips
   - **练习模式**: 按照requirements进行练习，计时+记录
   - **考核模式**: 完成challenge类型单元

**核心组件**:
```tsx
// 训练等级选择
<LevelSelector levels={trainingLevels} />

// 训练单元展示
<TrainingUnitCard unit={currentUnit} />

// 训练执行界面
<TrainingSession
  unit={currentUnit}
  mode={learningMode | practiceMode | challengeMode}
  onComplete={handleComplete}
/>
```

#### 2.2 创建 `targeted-practice.tsx` (针对性练习)

**功能模块**:

**特训模式**:
```tsx
<SpecialTrainingSelector>
  - 从67个训练单元中选择
  - 按技能分类筛选
  - 按难度筛选
  - 收藏常练单元
</SpecialTrainingSelector>
```

**自由训练**:
```tsx
<FreeTrainingMode>
  - 自定义练习内容
  - 简单计时器
  - 手动记录进度
  - 添加训练笔记
</FreeTrainingMode>
```

#### 2.3 更新导航菜单

在 `Header.tsx` 中:

### 阶段三: API集成 (2-3天)

#### 3.1 后端API已有

- ✅ `GET /api/training/levels` - 获取所有等级
- ✅ `GET /api/training/levels/:levelId` - 获取等级详情
- ✅ `GET /api/training/units/:unitId` - 获取单元详情
- ✅ `POST /api/training/progress/start` - 开始训练
- ✅ `POST /api/training/progress/update` - 更新进度
- ✅ `POST /api/training/progress/complete` - 完成训练

#### 3.2 需要新增的API

**获取推荐单元** (智能推荐):
```typescript
GET /api/training/recommended
Response: {
  units: TrainingUnit[],
  reason: string  // 推荐理由
}
```

**获取用户薄弱环节**:
```typescript
GET /api/training/weak-points
Response: {
  skills: string[],  // 薄弱技能
  suggestedUnits: TrainingUnit[]
}
```

## 📱 用户体验流程

### 新用户首次使用

```
1. 注册登录
   ↓
2. 查看【闯关系统】了解整体难度
   ↓
3. 进入【系统训练】开始Level 1
   ↓
4. 按顺序完成训练单元
   ↓
5. 完成一个Level后解锁下一Level
   ↓
6. 使用【针对性练习】强化薄弱环节
```

```

   ↓
2. 引导页介绍新功能
   ↓
3. 查看新的8级训练体系
   ↓
4. 继续在当前等级训练
   ↓
5. 探索针对性练习功能
```
