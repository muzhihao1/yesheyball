# 90天挑战课程映射方案

基于傅家俊台球十大招框架，将现有52集课程扩展到90天训练体系。

## 一、傅家俊台球十大招框架

### 第1招：基本功
- **动作**：手架、站位、握杆、入位、姿势
- **出杆**：定点、运杆、前停、回杆、推杆纠偏

### 第2招：发力
- **原理**
- **方法**：打点推接
- **控制**：大力、中力、小力

### 第3招：高效五分点
- **稳定性**：入位误差、视觉误差、出杆误差、身体控制、节奏控制
- **训练法**：大力定杆、中力定杆、小力定杆、高杆双下、低杆双下

### 第4招：准度
- **瞄准**：原理、平行法、度数法、重合法
- **打准**：大角度、小角度、中袋、底袋、半台、远台、母球贴库、子球贴库

### 第5招：杆法
- **基础**：高杆、中高杆、中杆、中低杆、低杆、小低杆、挫杆、推杆、低推、斯登
- **加塞**：顺塞、反塞、软塞、推塞、挫塞、接塞

### 第6招：分离角
- **原理**
- **方法**：高杆、中杆、低杆、大力、中力、小力
- **控制**：差点瞄准、吃库训练

### 第7招：走位
- **方法**：点、线、面
- **训练**：不吃库走位、一库走位、两库走位、多库走位

### 第8招：轻松清蛇彩
- **打法**：蛇彩一圈一、蛇彩一圈二、蛇彩三圈四、中袋四球赛
- **球型**：长台蛇彩、横台蛇彩、L形蛇彩、短库蛇彩

### 第9招：技能
- 开球、翻袋、解球
- 跳球、架杆、传球
- 借力、克拉克、贴球、扎杆

### 第10招：思路
- **进攻**：局面分析、局面处理
- **防守**：攻防转换、防守套路
- **心态**：球型训练、对抗、比赛

## 四、能力维度映射规则

### 主要能力维度 (primarySkill) 分配原则：

1. **accuracy (准度)**: 瞄准、进球、基本功相关
   - 第1招：基本功（手架、站位、出杆）
   - 第3招：高效五分点
   - 第4招：准度

2. **spin (杆法)**: 旋转、杆法、加塞相关
   - 第5招：杆法（高低杆、加塞）
   - 部分第6招：分离角的杆法控制
   - 第9招：技能中的传球、克拉克

3. **positioning (走位)**: 位置控制、走位相关
   - 第7招：走位
   - 第8招：轻松清蛇彩

4. **power (发力)**: 力量控制相关
   - 第2招：发力
   - 第3招：五分点的力量训练
   - 第9招：技能中的开球、跳球

5. **strategy (策略)**: 战术、思路、局面判断
   - 第6招：分离角原理
   - 第9招：技能中的解球、借力
   - 第10招：思路

### 训练记录方式 (scoringMethod)：

- **success_rate**: 准度相关训练（第1、3、4招）
  - 记录总次数和成功次数
  - 计算成功率更新 accuracy_score

- **completion**: 技巧和策略训练（第5、6、7、8、9、10招）
  - 按难度加权计算完成度
  - 更新对应维度的能力分

## 七、注意事项

1. **课程连贯性**：前52天保持原有课程的教学连贯性
2. **难度递进**：从基础到高级，循序渐进
3. **能力平衡**：确保5个维度都有充分训练
4. **实战导向**：第53天后加强实战和综合训练
5. **数据准确**：primary_skill 标注要准确，影响能力分计算
